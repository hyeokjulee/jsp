form tag

서버로 데이터를 전송하기 위해 사용
  • GET 방식
    • URL 주소 뒤에 파라미터를 붙여서 데이터 전달
    • 길이 제한 - 최대 2048자(4096바이트)
    • 북마크, 뒤로가기 가능
    • 보안 X
  • POST 방식
    • HTTP Request 헤더에 포함시켜서 데이터 전달
    • 길이 제한 X
    • 북마크, 뒤로가기 X
    • 보안 O
  • 속성
    action : 이동 위치 URL
    method : 데이터 전송 방식 get(기본) | post
    name : 폼의 이름 지정
    id : 폼의 아이디 지정
